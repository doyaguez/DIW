# README

## Introduction

En este documento realizamos una demostración en la que creamos un banner desde cero.    

## Code Samples

Lo primero que haremos sera buscar/crear las imágenes que vayamos a utilizar, en este caso hemos cogido una imagen de un grifo y la señal de prohibido junto a un 
texto que nos informa de que no desperdiciemos el agua, simplemente con arrastrarla hasta el programa que estamos utilizando en nuestro caso google web designer 
se nos añadirá a la biblioteca de imágenes, una ve hecho esto hemos creado un texto, para ello hemos pulsado el botón de añadir texto y hemos escrito pincha aquí.
Ahora pasaremos a darle un poco de dinamismo, para ello hemos cogido hemos arrastrado la imagen hasta el cuadro blanco que tenemos y un poco mas abajo en linea de 
tiempo hemos pinchado en 1,5 segundos con el botón derecho y hemos marcado la opción de crear evento. Hemos hecho lo mismo para el texto introducido así aparecerá 
el texto y la imagen moviéndose. Una vez que ya tenemos esto vamos a decirle que el texto sea un botón de imagen , para ellos pulsamos en Componentes y le decimos 
que sea un botón de imagen, nos saldrá un recuadro que tendremos que colocar encima del texto. Una vez creado esto en la pestaña de evento pulsaremos el cuadrado 
con el mas dentro y crearemos un evento. Nos saldrá una ventana en la que pulsaremos ratón>click>linea de tiempo y pulsaremos ir a la pagina y buscaremos nuestra 
pagina correspondiente, en este caso no hay nada así que crearemos otra pagina en la que arrastrando una imagen como anteriormente hemos explicado crearemos la un 
dinamismo al pinchar en el botón que hemos configurado.


## Installation

Lo único que hemos necesitado ha sido la instalación del programa Google Web Designer.
